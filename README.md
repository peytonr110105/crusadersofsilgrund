# Crusaders of Silgrund

MMORPG Conversion for Vanilla Minecraft 1.12.2 clients.

To Use:
Insert files into a PaperSpigot 1.12.2 server.

Dependendencies:
- PaperSpigot
- Essentials
- Skript
- TusKe
- SkQuery
- QuarSk
- Skellett
- ChestRestock
- GiantCaves
- OpenTerrainGenerator
- OldCombatMechanics
- Bookshelves_v1.2
