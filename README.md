# Crusaders of Silgrund

MMORPG Conversion for Vanilla Minecraft 1.18.2 clients.

To Use:
Insert files into a PaperSpigot or Pufferfish 1.18.2 server.

Dependencies:
- PaperSpigot / eqivelant
- Essentials
- Skript
- TusKe
- SkQuery
- QuarSk
- Skellett
- SkBee
- SkRayFall
- SkQuery
- ChestRestock
- GiantCaves
- OpenTerrainGenerator
- OldCombatMechanics
- Bookshelves_v1.2
